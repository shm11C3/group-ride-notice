# Bipokele

## 概要

Bipokele は自転車乗りが走行会・練習会を主催、または走行会・練習会に参加することができるWEBアプリケーションです。

### URL
https://bipokele.com

## 使用技術

- サーバーサイド
  - PHP 8 (Laravel 8, Laravel-admin, socialite) 
 
- クライアントサイド
  - HTML(Blade)
  - JavaScript (Vue.js 2.6, JQuery)
  - SCSS (Bootstrap-honoka 4.4)

- インフラ 
  - Nginx
  - MySQL
  - PHPMyAdmin(開発環境時)
